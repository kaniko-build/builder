# kaniko-build/build

Repository to build OCI images of [chainguard-dev fork of kaniko](https://github.com/chainguard-dev/kaniko/)

Actions are taken from the original build process with few changes to adapt to ghcr and monkey patch version in `Makefile`.

Results are available at:

* Warmer: <https://github.com/kaniko-build/builder/pkgs/container/dist%2Fchainguard-dev-kaniko%2Fwarmer>
* Executor: <https://github.com/kaniko-build/builder/pkgs/container/dist%2Fchainguard-dev-kaniko%2Fwarmer>
